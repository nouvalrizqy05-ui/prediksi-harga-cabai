## Overview

This repository contains the complete replication package for the above article.
It includes:

- **Raw data** from PIHPS (National Strategic Food Price Information Center)
- **Processed CSVs** with train/test split labels
- **Full Python analysis pipeline** producing all metrics and figures
- **Pre-generated figures** (300 DPI PNG) as published in the article

The study compares Simple Linear Regression (SLR) and Multiple Polynomial
Regression (MPR) — using national aggregate price as an exogenous variable —
to predict retail chili prices in Central Java for four commodity variants:
Large Red Chili (CMB), Curly Red Chili (CMK), Green Bird's Eye Chili (CRH),
and Red Bird's Eye Chili (CRM).

## Data Description

### Source

All price data were obtained from the **National Strategic Food Price
Information Center (PIHPS)**, operated by Bank Indonesia and the Indonesian
Ministry of Industry. URL: https://pihps.kemenperin.go.id

### Coverage

| Field         | Value                                    |
|---------------|------------------------------------------|
| Period        | July 2022 – December 2025                |
| Frequency     | Monthly (first available trading day)    |
| Observations  | 42 per commodity                         |
| Geography     | Jawa Tengah (Y) and All Provinces (X)    |
| Unit          | IDR per kg (Indonesian Rupiah per kg)    |

### Variables

| Column                  | Description                                          |
|-------------------------|------------------------------------------------------|
| `date`                  | ISO 8601 date (YYYY-MM-DD)                           |
| `harga_nasional_idr_kg` | National aggregate retail price — exogenous var (X)  |
| `harga_jateng_idr_kg`   | Central Java retail price — target variable (Y)      |
| `split`                 | `train` (n=33, Jul 2022–Mar 2025) / `test` (n=9, Apr–Dec 2025) |

### Commodity Codes

| Code | Full Name (ID)       | Full Name (EN)              |
|------|----------------------|-----------------------------|
| CMB  | Cabai Merah Besar    | Large Red Chili             |
| CMK  | Cabai Merah Keriting | Curly Red Chili             |
| CRH  | Cabai Rawit Hijau    | Green Bird's Eye Chili      |
| CRM  | Cabai Rawit Merah    | Red Bird's Eye Chili        |

## Reproducing the Results

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

Tested on Python 3.10. Key versions: numpy 1.24+, scikit-learn 1.3+,
matplotlib 3.7+, openpyxl 3.1+.

### 2. Run the full pipeline

```bash
python src/analysis.py
```

This will regenerate all figures in `figures/` and all CSVs in `results/`.
Expected runtime: < 60 seconds on any modern laptop.

### 3. Expected output (results/metrics_all.csv, test set)

| Commodity | Model  | MAPE (%) | Rating    |
|-----------|--------|----------|-----------|
| CMB       | SLR    | 10.12    | Good      |
| CMB       | Poly-2 | **9.62** | Very Good |
| CMK       | SLR    | 11.70    | Good      |
| CMK       | Poly-2 | **11.14**| Good      |
| CRH       | SLR    | 10.42    | Good      |
| CRH       | Poly-2 | **8.94** | Very Good |
| CRM       | SLR    | 17.72    | Good      |
| CRM       | Poly-3 | **17.07**| Good      |

MAPE thresholds: < 10% = Very Good, 10–20% = Good, 20–50% = Fair, > 50% = Poor.

## Methodology Summary

- **Train/test split:** 80:20 temporal split (33 train, 9 test; no shuffling)
- **Predictor (X):** National aggregate PIHPS price (IDR/kg), centered before polynomial expansion to reduce VIF
- **Target (Y):** Central Java retail price (IDR/kg)
- **Models compared:** SLR, Poly-2, Poly-3, Poly-4, Poly-5
- **Degree selection:** Post-hoc by minimum test-set MAPE
- **Metrics:** MAE, RMSE, MAPE
- **OLS assumptions verified:** Pearson r (linearity), Breusch-Pagan (homoscedasticity), VIF on centered features (multicollinearity)
